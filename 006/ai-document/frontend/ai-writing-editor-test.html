🏗️ 技术架构

前端技术栈

  • React 18 + TypeScript
  • Ant Design UI组件库
  • React Router 路由管理
  • Quill 富文本编辑器
  • EventSource 流式数据
  • CSS3 响应式设计

后端技术栈

  • FastAPI + Python
  • AutoGen 0.6.1 智能体框架
  • SQLAlchemy ORM
  • MySQL 数据库
  • Server-Sent Events
  • DeepSeek AI模型

🎮 快速体验

💡 使用提示

  • 在编辑器中输入 @ 符号可以快速调用AI写作助手
  • 选择合适的写作主题，填写详细信息可以获得更好的生成效果
  • 支持实时流式输出，内容会直接在编辑器中显示
  • 如果后端服务不可用，系统会自动使用模拟数据进行演示
