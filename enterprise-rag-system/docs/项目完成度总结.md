## 🚀 部署指南

### 1. 环境要求
- Docker 20.10+
- Docker Compose 2.0+
- 8GB+ 内存
- 50GB+ 存储空间

### 2. 快速部署
```bash
# 设置环境变量
export QWEN_API_KEY="your_qwen_api_key"
export SECRET_KEY="your_secret_key"
export DOMAIN_NAME="yourdomain.com"

# 运行部署脚本
chmod +x scripts/deploy.sh
./scripts/deploy.sh
```
